自由意志

发表于 2019-07-16   |  

我还记得小时候的我，第一次有长大的感觉，是在一次考试失利，不能去好班之后的那半年。
经历了非常阴暗的一段时期，成绩虽然不太稳定但也是考过很多次年级前几的水平，竟然被留在了烂班里，很是抬不起头来。每天哭泣，但没想到最后居然靠着几本名著活了过来。现在是非常感谢那次失利的。莫名多了际遇，可以与一些坚强的、立体的、明确的灵魂进行跨时空的交流，是那样奇妙的馈赠。

恢复过来后，虽然在烂班里，但开始和以前一些从未接触过的人一起学习，骑车回家。甚至一些霸凌过我的人，也向他们释放善意，与他们嬉戏打闹。那次考试之后的春天，还有一次补漏考试，我发挥自觉还可以，却没有到可以把我送进好班的程度，得知名次后不禁有些愕然。回家的路上，唐小胖对我说：“田姐你还是很厉害的。”我回问道：“你真的这么觉得吗？”他说对啊，眼里竟然还有扑闪扑闪的光。你可能不知道唐小胖是谁，这么讲吧，他曾经是个坏人，在我心里。一个在我初二时候发育过早带着大家给我取外号的人。嘴巴贱到若是我有脾气，可能一个耳光能抽上去。可惜我曾是个软弱的人，并不怎么和人硬钢。这种后悔的劲儿还没来得及过，却听到他给我真诚的鼓励，心里惊讶到不行。

但是从那以后便积累了些自由意志的底魄。

要拥有自由意志，你得坚信凭借自己，是可以战胜他人，可以抹去记忆，可以扭转地位的。若是心里把自己和周遭联系的太过紧密，难免会在做决定时顾及这个，顾及那个。你若是希望有自由意志，便得明白人的印象并不可靠，而且随时都可以变化。你在进行社交的时候可以表现得像个社会人，尊重他人的习惯，但是在你独立思考或者说要做出对自己非常关键的决定时，你得将他人的印象抛向一边，抓紧自己的自由意志，绝不让一分田地。
需要解释的是，在一些大的问题上，例如色情和暴力，你并不能只凭借自己的想法去做决定，因为这已经超过个人意志的界限了，你需要一个相对稳定的社会身份去做所有的事情。
有一件事情是值得庆幸的。这世界上，有许多许多的事情都在界限之内，意味着你可以尽情、肆意、毫无顾忌，比如剪个超短的头发，穿新的风格的衣服，拥有奇特的癖好并且和同好交流，你都不用考虑别人的感受。认清界限很重要，但它只是个必要条件。

切勿遗忘心中的自由意志，这才是要铭记在心的。

cameron boyce
